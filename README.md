# Light-up-the-room
เนื่องด้วยสถานการ์ณการแพร่ระบาดของโรค COVID-19 ส่งผลให้วิถีชีวิตของคนทั่วโลกเปลี่ยนไปเป็นอย่างมาก ยกตัวเช่น การเว้นระยะห่าง การรักษาความสะอาด การใส่หน้ากากอนามัย และถึงแม้ว่าสถานการ์ณการแพร่ระบาดของโรค COVID-19 จะมีแนวโน้มที่ดีขึ้นตามลำดับ แต่มีความเป็นไปได้สูงว่าหลังจากโรคระบาดสงบลงแล้ว วิถีชีวิตของคนทั่วโลกไม่กลับไปเป็นเหมือนช่วงยุคก่อนโรคระบาดอีกต่อไป หรือที่เรียกว่าวิถีชีวิตแบบ **new normal** นั่นเอง
